Function In JavaScript

In JavaScript, a function is a value, so we can deal with it as a value/ We first define the function and then assign it to a variable unlike in expressions

JavaScript functions are a special type of objects, called function objects. A function object includes a string which holds the actual code i.e, the function body of the function.

Functions can be written more compactly with the help of Arrow functions. Arrow function expressions are ill suited as methods, and they cannot be used as constructors.
